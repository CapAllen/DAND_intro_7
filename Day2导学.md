## 1.算术

这里表面看只有算术，其实，你在不知不觉间还学到了：

**print函数**

- python的一个重要内置函数`print`，也就是“打印”的意思啦，你想把你的算数结果进行输出的时候，就会用到它。
- 使用方法：print(你想要输出的表达式)。注意：不要忘了加小括号哦。

**算术运算符**

假设 a = 12, b = 26

| 运算符 | 描述                                           | 实例                                                   |
| ------ | ---------------------------------------------- | ------------------------------------------------------ |
| +      | 加，数字就是做加法/字符串就是拼接              | a + b 输出结果 28                                      |
| -      | 减，做减法                                     | a - b 输出结果 -28                                     |
| *      | 乘 ，数字的话就是相乘/字符串的话就是重复若干次 | a * b 输出结果  312                                    |
| /      | 除，做除法                                     | b / a 输出结果  2.1666666666666665（自动转化为浮点型） |
| %      | 取模 ， 即做除法后的余数                       | b % a 输出结果 2                                       |
| **     | 幂                                             | a**b 为12的26次方                                      |
| //     | 取整，即做除法后的商                           | 9//2 输出结果 4 , 9.0//2.0 输出结果 4.0                |

**python代码规范**

想要学好一门语言，语法作为语言中**法律**一样的存在，是非常重要的，所以请大家一定要恪守这个法律，在初学阶段打好基础，不要养成不良的编码习惯。

- 空格。在较低优先级的运算符两端加上一个空格，方便阅读即可，示范：

  ```python
  print(4/2 - 7*7)
  ```

- 注释。注释表示的是对某一行代码的解释，并不会被执行。python中单行注释以 **#** 开头，示例：

  ```python
  # Write an expression that calculates the average of 23, 32 and 64.
  # Place the expression in this print statement.
  print((23+32+64)/3)
  ```

- 大小写敏感。在python中区分大小写，所以a与A并不一样，print也不能写成Print。

## 2.整数和浮点数

整数与浮点数都属于python中的一种数据类型。

整数比较好理解，就是数学上学的整数的意思，用int表示；

浮点数是一种储存小数的形式，用float表示。

在python中，两个整数相除的话，会自动生成浮点型的数据，示例：

```python
a = 12
b = 26
print(b/a)
```

结果为：

```python
2.1666666666666665 
```

## 4.变量

使用变量可以更高效地说明数据的转换，在python中定义变量的方式也十分简单，只需要在变量名和赋值之间加个`=`即可。

示例：

```python
a = 13 #定义一个变量a，其值为13
```

**注意**：

- 变量再次赋值时，会覆盖掉之前的赋值结果，示例：

  ```python
  a = 13
  a = 7
  print(a)
  ```

  其结果为：7

- 区分大小写，a与A表示的是两个不同的变量。

## 5.比较运算符

| 运算符 | 描述                                                         |
| ------ | ------------------------------------------------------------ |
| ==     | 等于；比较对象是否相等                                       |
| !=     | 不等于； 比较两个对象是否不相等                              |
| >      | 大于 ； 返回x是否大于y                                       |
| <      | 小于 ；返回x是否小于y。所有比较运算符返回1表示真，返回0表示假。这分别与特殊的变量True和False等价。注意，这些变量名的大写。 |
| >=     | 大于等于；返回x是否大于等于y。                               |
| <=     | 小于等于；返回x是否小于等于y。                               |

这里值得注意的是：

- 等于是`==`，赋值是`=`，千万不要搞混了；
- 比较运算符的返回结果为“是”或者“否”，在python中即为“True”或者“False”，这就是**布尔值**啦。

## 6.字符串

字符串即为一个字符的序列，是文本的一种呈现形式，也是python的数据类型之一。

它的定义也很简单，只要放在单引号或双引号里就可以了，当然，前后引号要对应。

这里要注意的一点是：如果想在引号中使用引号，比如说下面原来的文本中就存在引号时该怎么处理。

```python
I think you're an encyclopaedia salesman。#原文本
```

这里提供两种方法，一种是使用反斜杠`\`，放在文本中原本存在的引号前面

```python
str_a = 'I think you\'re an encyclopaedia salesman'#将这段文本定义为变量str_a
```

另外一种就是使用不同于原文中存在的引号

```python
str_a = "I think you're an encyclopaedia salesman"#将这段文本定义为变量str_a
```
